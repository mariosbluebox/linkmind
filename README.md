# LinkMind

A personal knowledge vault for tools, repos, and ideas found on X, GitHub, and elsewhere. Designed to be read and maintained by Claude Code.

## Structure

```
linkmind/
├── CLAUDE.md          # Instructions for Claude Code
├── README.md          # This file
├── TEMPLATE.md        # Master template (never edit this)
├── INDEX.md           # Catalogue of all entries
├── CONNECTIONS.md     # How entries relate to each other
└── entries/           # Individual knowledge entries
    └── *.md
```

## Quick workflow

1. Find something interesting on X or GitHub
2. Open Claude Code in this folder
3. Say: "Add this to LinkMind: [URL]" and explain why it caught your eye
4. Claude Code creates the entry, updates the index, and checks for connections

## Asking LinkMind for help

- "What tools do I have for building data pipelines?"
- "What connects to [tool name]?"
- "I need to build X — what in my vault could help?"
- "Show me all entries tagged with [tag]"
