# LinkMind — Connections Map

> This file tracks how entries relate to each other. It's what turns a flat list of bookmarks into a knowledge graph. Claude Code maintains this file as new entries are added.

## How to read this
Each connection describes a relationship between two or more entries. The format:
- **What connects**: Which entries are linked
- **How they connect**: The specific relationship (feeds into, replaces, complements, enables, etc.)
- **Potential combo**: What you could build or achieve by combining them
